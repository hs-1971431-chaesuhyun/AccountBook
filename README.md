https://youtu.be/5gHKr1R_s6Q

## 기능 목록

- 월별 수입과 지출 관리
- 각 날짜별 수입 및 지출 기록
- 수정 및 삭제 기능 제공

## 사용 방법

1. 앱을 실행하고 원하는 날짜를 선택합니다.
2. 수입과 지출 내역을 추가하거나 수정합니다.
3. 각 날짜별 수입과 지출을 확인하고 관리합니다.
